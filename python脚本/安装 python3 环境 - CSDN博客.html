3.添加环境变量

有人会问什么是环境变量呢？

在这里我解释一下，环境变量，其实是在cmd命令窗口中，执行某个命令的时候，系统会自动索引到该命令的目录下去调用这个命令工具。 例如：

我已经安装完成了python，但想要在cmd 命令窗口下编程 


系统是找不到python是什么的，除非我们直接去安装python的目录下，找到python.exe(命令其实就是一个应用程序)


看，这样就运行成功了，，但我们总不能每次想要执行python 都是该目录下找python.exe吧、我希望直接在CMD命令下敲python 系统就能自动索引到该目录去执行，那么环境变量就起到重要重要了

右键计算机-属性-高级系统设置-高级-环境变量

系统变量 -找到变量是PATH -编辑 -并在后面添加python.exe的目录

%SystemRoot%\system32;%SystemRoot%;%SystemRoot%\System32\Wbem;%SYSTEMROOT%\System32\WindowsPowerShell\v1.0\;C:\Program Files\MySQL\MySQL Server 5.5\bin;E:\MySql\mysql-5.6.24-win32\bin;C:\Users\Administrator\AppData\Local\Programs\Python\Python37-32
